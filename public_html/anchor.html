HTML anchor


An HTML anchorThe HTML anchor tag defines a hyperlink that links one page to another page. The "href" attribute is the most important attribute of the HTML a tag. href attribute of HTML anchor tag The href attribute is used to define the address of the file to be linked. In other words, it points out the destination page. The syntax of HTML anchor tag is given below. Link Text

Let's see example of html anchor tag

<a href="second.html">Click for Second Page</a>
