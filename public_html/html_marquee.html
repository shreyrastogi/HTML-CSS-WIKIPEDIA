<Marquee>




An HTML marquee is a scrolling piece of text displayed either horizontally across or vertically down your webpage depending on the settings.



Syntax


The <marquee> element comes in pairs. It means that the tag has opening (<marquee>) and closing (</marquee>) elements.




Notes:


The <marquee> element is a non-standard HTML element. If you use this element, your pages or apps may be broken.





HTML<Marquee> tags


The marquee is a non-standard HTML tag which was used to create a scrolling text or an image.

It was used to make the text/image scroll horizontally across or vertically down the web page.



Attributes:


behaviour, bgcolor, direction, height, width, hspace, loop, vspace etc.

EXAMPLE

<!DOCTYPE html>
<html>
<head>
<title>marquee</title>
</head>
<body>

<marquee>this is an example
</marquee>

</body>
</html>

OUTPUT

this is an example
