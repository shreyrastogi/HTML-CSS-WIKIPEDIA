<Textarea>




The <textarea> element is used to create a text input area of unlimited length.



Definition and Usage


The <textarea> tag defines a multi-line text input control.

The <textarea> tag defines a multi-line text input control.

The size of a text area can be specified by the cols and rows attributes, or even better; through CSS' height and width properties.



Attributes:

autfocus, cols, rows, dirname, disabled, maxlength, name, required, wrap etc.



EXAMPLE

<!DOCTYPE html>
<html>
<head>
<title>textarea</title>
</head>
<body>

<textarea rows="4" cols="50"> This is an example of Textarea. </textarea> </body>
</html>
