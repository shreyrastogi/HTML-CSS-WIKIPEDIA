HTML Computer Code



The HTML <CODE> element defines a fragment of computer code.

Text surrounded by <CODE> tags is typically displayed in the browser's default monospace font:

Examples in Every Chapter


Notice that the <CODE> element does not preserve extra whitespace and line-breaks.
To fix this, you can put the <CODE> element inside a <pre> element:



EXAMPLE

<!DOCTYPE html><br> <html> <body> <h2>Computer Code</h2> <p>Some programming code:</p> <code> x = 5;<br> y = 6;<br> z = x + y; </code> </body> </html>

OUTPUT

The code Element

Programming code example: x = 5; y = 6; z = x + y;
