HTML NOSCRIPT TAG



HTML <noscript> tag is used to define an alternate content for the users who have disabled
scripting from the browser and want to access the web page.
The <noscript> element can be used within <head> and <body> tags.

While using noscript tag inside <head> element, <noscript> must contain <link>, <style>, and <meta> tags.

The text inside the <noscript> element will be displayed if the user's browser is not script supporting.

EXAMPLE

<script>
document.write("Welcome to JavaTpoint")
</script>
<noscript>Sorry! Your browser does not support JavaScript.!&ht;/noscript>

OUTPUT

Welcome to JavaTpoint
