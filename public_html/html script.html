HTML SCRIPT TAG




HTML <script> tag is used to define an alternate content for the users who have disabled
scripting from the browser and want to access the web page.
The <noscript> element can be used within <head> and <body> tags.

JavaScript is used for image manipulation, form validation, and dynamic content.
The syntax of script tag is given below:


EXAMPLE

<script>type="text/javascript"
function msg(){
alert("Welcome to JavaTpoint") }
</script>

OUTPUT

Welcome to JavaTpoint


Link script file

The script tag can be used to link external script file by src attribute.
It must be used within the <head> tag only.


EXAMPLE

<script>type="text/javascript"src="message.js"/>

</script>
