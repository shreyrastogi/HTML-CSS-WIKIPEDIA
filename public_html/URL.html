HTML Uniform Resource Locators



A URL is another word for a web address.

Most people enter the name when surfing, because names are easier to remember than numbers.

URL - Uniform Resource Locator


Web browsers request pages from web servers by using a URL.

A Uniform Resource Locator (URL) is used to address a document (or other data) on the web.



EXAMPLE


Explanation:- 1.scheme - defines the type of Internet service (most common is http or https) 2.prefix - defines a domain prefix (default for http is www) 3.domain - defines the Internet domain name 4.port - defines the port number at the host (default for http is 80) 6.path - defines a path at the server (If omitted: the root directory of the site) 7.filename - defines the name of a document or resource
